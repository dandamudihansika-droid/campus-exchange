# 🚀 Quick Deploy - Get Your Campus Exchange Online NOW!

## 📋 3-Minute Deployment Guide

### Option 1: Netlify (Easiest) ⭐

#### Step 1: Go to Netlify
1. Open your browser → [netlify.com](https://netlify.com)
2. Click "Sign up" → Continue with GitHub

#### Step 2: Drag & Drop Deploy
1. Click "Drag and drop your site output folder here"
2. Select your ENTIRE project folder: `field project`
3. Drag it onto the Netlify page
4. Wait 30 seconds...

#### Step 3: Your Site is LIVE! 🎉
- Netlify will give you a random URL like: `amazing-johnson-123456.netlify.app`
- Your Campus Exchange is now online!

---

### Option 2: GitHub Pages (Free & Easy)

#### Step 1: Create GitHub Account
1. Go to [github.com](https://github.com) → Sign up
2. Verify your email

#### Step 2: Upload Your Files
1. Click "+" → "New repository"
2. Repository name: `campus-exchange`
3. Click "Create repository"
4. Click "uploading an existing file"
5. Drag your entire `field project` folder
6. Click "Commit changes"

#### Step 3: Enable GitHub Pages
1. In your repository, click "Settings"
2. Scroll down to "Pages"
3. Source: "Deploy from a branch"
4. Branch: "main" → "/ (root)"
5. Click "Save"

#### Step 4: Your Site is Live! 🎉
- Go to: `https://YOUR_USERNAME.github.io/campus-exchange`
- Wait 2-3 minutes for it to activate

---

### Option 3: Vercel (Most Professional)

#### Step 1: Go to Vercel
1. Open browser → [vercel.com](https://vercel.com)
2. Click "Sign up" → Continue with GitHub

#### Step 2: Import Repository
1. Click "New Project"
2. Connect your GitHub account
3. Select your `campus-exchange` repository
4. Click "Import"

#### Step 3: Deploy
1. Framework: "Other"
2. Click "Deploy"
3. Wait 1 minute...

#### Step 4: Your Site is Live! 🎉
- URL: `https://campus-exchange.vercel.app`
- Professional URL with HTTPS!

---

## 🔧 Quick Fix for Online Deployment

### Update Image Paths (Important!)
Before deploying, make sure images work online:

In `data.js`, change all image paths to start with `/`:
```javascript
// Change this:
image: "images/book.jpg"

// To this:
image: "/images/book.jpg"
```

### Update API Base URL (if needed)
In `js/api.js`, make sure this line exists:
```javascript
const API_BASE_URL = window.location.origin;
```
